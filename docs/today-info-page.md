# 오늘의 정보 페이지

## 개요

`/dashboard/air-quality` 경로에 위치하며, 대기질·날씨·교통 정보를 하나의 페이지에서 탭으로 전환하여 확인합니다. 모바일 최적화 UI입니다.

## 탭 구성

| 탭 | 키 | 데이터 소스 | 지원 도시 |
|----|-----|-----------|---------|
| 🌫️ 대기질 | `air` | 한국환경공단 에어코리아 | 서울·부산·대구·인천·광주·대전·울산·경기·강원·충북·충남·전북·전남·경북·경남·제주 |
| 🌤️ 날씨 | `weather` | 기상청 단기예보 | 서울·부산·대구·인천·광주·대전·울산·제주 |
| 🚌 교통 | `transit` | 국토교통부 버스노선 | 부산·대구·인천·광주·대전·울산·경기 |

## UI 구조

```
[상단 pill 탭 셀렉터]
  ┌────────────────────────────────┐
  │  🌫️ 대기질  🌤️ 날씨  🚌 교통   │
  └────────────────────────────────┘

[탭별 콘텐츠]
  ┌────────────────────────────────┐
  │  [도시 선택 셀렉트]              │
  │  [차트 / 요약 카드]              │
  │  [목록 카드]                    │
  └────────────────────────────────┘
```

### 탭 디자인 스펙
- 배경: `rgba(99,102,241,0.06)` (인디고 틴트)
- 활성 탭: 흰색 배경 + 그림자 pill
- 전환 애니메이션 없음 (즉시 교체)

## 컴포넌트 구조

```
page.tsx
├── CitySelect      공통 도시 선택 셀렉트
├── Skeleton        로딩 스켈레톤
├── ErrorBox        에러 메시지 박스
├── AirTab          대기질 탭
│   ├── 막대 차트 (Recharts BarChart)
│   └── 측정소별 카드 목록
├── WeatherTab      날씨 탭
│   ├── 요약 카드 4개 (기온·강수·습도·풍속)
│   ├── 기온 라인 차트 (Recharts LineChart)
│   └── 시간대별 예보 카드
└── TransitTab      교통 탭
    ├── 버스 노선 카드 목록
    └── 페이지네이션
```

## 대기질 등급 색상

| 등급 | 텍스트 | 의미 |
|------|-------|------|
| 1 | 좋음 | 파란색 |
| 2 | 보통 | 초록색 |
| 3 | 나쁨 | 주황색 |
| 4 | 매우나쁨 | 빨간색 |

`getAqiGrade(value)` 유틸(`src/lib/utils.ts`)로 수치 → 등급 변환

## 데이터 훅

| 훅 | 파일 | SWR 키 |
|----|------|--------|
| `useAirQuality(city)` | `src/hooks/use-air-quality.ts` | `/api/air-quality?city=...` |
| `useWeather(city)` | `src/hooks/use-weather.ts` | `/api/weather?city=...` |
| `useTransit(city, page)` | `src/hooks/use-transit.ts` | `/api/transit?city=...&page=...` |
